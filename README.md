# Todo Schedule

Flutter-приложение для управления задачами с расписанием по дням и неделям.

## Возможности

- Расписание по дням с календарём
- Недельное расписание с повторяющимися задачами
- Повторение: ежедневно, еженедельно, каждые N недель
- Указание длительности и времени начала
- Приоритеты (низкий / средний / высокий)
- Цветовые категории
- Тёмная и светлая тема
- Красивые анимации

## Сборка

### Требования
- [Flutter SDK](https://docs.flutter.dev/get-started/install) 3.2+

### Быстрый старт
```bash
# Windows — двойной клик на build.bat
# Или из терминала:
flutter create . --platforms=android,windows
flutter pub get
flutter run -d windows
```

## Платформы
- Android
- Windows
